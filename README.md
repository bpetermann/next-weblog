# NextBlog

## Project Overview

### Description

- Blogpost project built with nextjs
- Using getstaticprops, next/image, etc.
- [View Live on Vercel](https://next-weblog.vercel.app/)

## Getting Started

```
git clone https://github.com/bpetermann/next-weblog.git
cd next-shopping-cart
npm install
npm run dev
# or
yarn dev
```
